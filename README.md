# Project Title

Project website
[Live Try](https://train-ticket-cr.up.railway.app/)

---
## Project Explanation
This project is focused on developing a website (mobile first design) to buy train tickets for the 3 principal routes in Costa Rica.

---
## Technology Stack
- **Frontend:** Angular JS, SCSS
- **Backend:** Java, Spring Boot,(Maven)
- **Database:** MongoDB
- **Deployment:** Railway, Versel
- **Version Control:** Git

---
## Features
- **QR Ticket info**
- Real routes based in INCOFER website.
- Real Schedules.
- Mock Payment page.
